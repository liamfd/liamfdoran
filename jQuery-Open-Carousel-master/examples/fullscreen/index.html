jQuery Open Carousel - Fullscreen Example

by: Justin McCandless
github: github.com/justinmc/jQuery-Open-Carousel



This example demonstrates the data-ocarousel-fullscreen configuration attribute, which dynamically sets each frame's width to the full width of the screen. It's also fully responsive; try changing the size of your browser window or rotating your mobile device and see how it adapts.
